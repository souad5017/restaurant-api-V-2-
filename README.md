Restaurant AI API
Description

This project is a RESTful API built with Laravel that allows users to:

Browse dishes (plates)
Define dietary preferences
Get AI-based food recommendations
Analyze nutritional compatibility
Features
Authentication with Laravel Sanctum
User profile & dietary preferences
Categories & Ingredients management
Plates management with image upload (Cloudinary)
AI Recommendation system (Groq / LLM)
Queue & Jobs for background processing
Admin statistics dashboard
API Documentation

API documentation is available here:

https://behr0d66w8.apidog.io

Technologies Used
Laravel
MySQL
Laravel Sanctum
Cloudinary (image upload)
Groq API (AI recommendations)
Queues & Jobs
